How to Use

1. Basic Embed

Simply add this script tag to your HTML:

<script src="https://mpestrax.vercel.app/player.js" data-src="https://example.com/video.mp4"></script>

2. With Custom Container

Wrap in a container to control size:

<div style="max-width: 800px; margin: 0 auto;">
  <script src="https://mpestrax.vercel.app/player.js" data-src="https://example.com/video.mp4"></script>
</div>

3. Custom Colors

Customize the player theme with your brand colors:

<script src="https://mpestrax.vercel.app/player.js"
  data-src="https://example.com/video.mp4"
  data-primary="#ff6b6b"
  data-accent="#4ecdc4"
  data-surface="rgba(255, 255, 255, 0.2)"
  data-text="#ffffff"
  data-outline="rgba(255, 255, 255, 0.3)"></script>

4. Multiple Players

Add multiple players with different themes:

<script src="https://mpestrax.vercel.app/player.js" data-src="video1.mp4" data-primary="#e74c3c"></script>
<script src="https://mpestrax.vercel.app/player.js" data-src="video2.mp4" data-primary="#3498db"></script>

Color Customization

Available Color Attributes
  • data-primary - Main accent color (progress bars, buttons)
  • data-accent - Secondary accent color (gradients)
  • data-surface - Background surface color
  • data-text - Text color
  • data-outline - Border and outline color
Popular Color Schemes

Ocean Blue: data-primary="#0077be"

Sunset Orange: data-primary="#ff6b35"

Forest Green: data-primary="#2d5a27"

Royal Purple: data-primary="#6a4c93"
